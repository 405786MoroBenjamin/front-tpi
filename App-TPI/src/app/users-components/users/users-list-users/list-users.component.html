Información sobre la Lista de Usuarios

En esta pantalla puedes visualizar la lista completa de usuarios con sus detalles principales. A continuación, algunas características clave:

  • La tabla está ordenada de manera predeterminada por fecha de creación en orden descendente
  • Columnas visibles en la tabla:
    • Fecha de creación: Muestra la fecha en la que se creó el usuario.
    • Nombre: El nombre completo del usuario.
    • Rol: El rol asignado al usuario, como administrador o usuario estándar.
    • Lote: Número de lote asociado al usuario; es la numeración, no la cantidad de lotes que posee.
  • Filtros disponibles para refinar los resultados:
    • Buscar: Utiliza el campo de búsqueda para encontrar usuarios por nombre o información relevante.
    • Rol: Filtra los usuarios por su rol asignado.
    • Rango de fecha: Filtra usuarios dentro de un rango específico de fechas de creación.
  • Puedes exportar la lista de usuarios en formato Excel o PDF usando los botones correspondientes.
  • Para restablecer todos los filtros, haz clic en el botón de reinicio de filtros ().
